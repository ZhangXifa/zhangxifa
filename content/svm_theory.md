# 支持向量机原理

## 1. 概述

本研究采用线性支持向量机（Linear Support Vector Machine, Linear SVM）进行脊柱点的识别。线性SVM是一种基于统计学习理论的监督机器学习算法，由Cortes和Vapnik首次提出[62]。其核心思想在于通过构造一个最优超平面，最大化不同类别样本之间的分类间隔，从而提升模型的泛化能力[63]。线性SVM在小样本、非线性及高维模式识别等问题上展现出良好的性能。其理论基础主要建立在两大支柱之上：一是结构风险最小化原则，通过平衡经验风险与置信范围，兼顾模型的拟合能力与泛化性能；二是VC维理论，深刻揭示了模型复杂度与学习能力之间的内在关系，为模型的有效性提供了理论保障。本研究采用LibLinear库实现线性SVM模型，该库针对大规模数据进行了优化，在保证分类精度的同时具备较高的计算效率。

## 2. 间隔与支持向量

线性SVM的核心在于寻找一个具有"最大间隔"的分类超平面。所谓"间隔"，指的是超平面与其两侧最近的样本点之间的距离，而那些决定间隔位置的样本点即为"支持向量"。因此，支持向量直接决定了最优超平面的位置，SVM模型的构建与性能表现均依赖于这些关键样本。

给定一个多维空间中的训练数据集 $D = \{(x_i, y_i) \mid i \in \{1,2,...,n\}\}$，其中 $x_i \in \mathbb{R}^p$ 表示样本特征，$y_i \in \{-1, +1\}$ 表示对应的样本标签。线性SVM的目标是在该空间中寻找一个能够有效区分不同类别样本的超平面。如图X所示，可能存在多个超平面均能正确划分训练样本，而SVM通过训练选择出使两类样本间"间隔"最大的最优超平面，以提升分类器在未知样本上的泛化能力。

超平面可由线性方程表示：

$$\omega^T x + b = 0 \tag{2.23}$$

其中 $\omega = \{\omega_1, \omega_2, ..., \omega_p\}$ 是决定超平面方向的法向量，$\omega^T$ 为其转置形式，$b$ 为偏置项，控制超平面与原点的偏移。样本点 $(x_i, y_i)$ 到超平面的距离可表示为：

$$r = \frac{|\omega^T x_i + b|}{\|\omega\|} \tag{2.24}$$

对于线性可分问题，存在超平面使得所有样本满足分类约束：

$$\begin{cases} \omega^T x_i + b \geq +1, & y_i = +1 \\ \omega^T x_i + b \leq -1, & y_i = -1 \end{cases} \tag{2.25}$$

该约束可统一写为 $y_i(\omega^T x_i + b) \geq 1$。如图X所示，距离超平面最近的样本点被称为"支持向量"，在这些支持向量处，式(2.25)取等号成立。两类支持向量到超平面的垂直距离（即"间隔"）为：

$$\gamma = \frac{2}{\|\omega\|} \tag{2.26}$$

因此，寻找最大间隔超平面的问题可转化为以下优化问题：

$$\begin{cases} \min\limits_{\omega,b} & \frac{1}{2}\|\omega\|^2 \\ \text{s.t.} & y_i(\omega^T x_i + b) \geq 1, i=1,2,...,n \end{cases} \tag{2.27}$$

其中最小化 $\frac{1}{2}\|\omega\|^2$ 等价于最大化间隔 $\frac{2}{\|\omega\|}$。这是一个凸二次规划问题，可通过拉格朗日乘子法求解。

## 3. 对偶问题与KKT条件

为求解式(2.27)的优化问题，引入拉格朗日函数：

$$\mathcal{L}(\omega, b, \alpha) = \frac{1}{2}\|\omega\|^2 + \sum_{i=1}^{n}\alpha_i(1 - y_i(\omega^T x_i + b)) \tag{2.28}$$

其中 $\alpha_i \geq 0$ 为拉格朗日乘子。根据拉格朗日对偶性，原始问题的对偶问题为极大极小问题：

$$\max_{\alpha}\min_{\omega,b}\mathcal{L}(\omega, b, \alpha)$$

首先求 $\mathcal{L}$ 对 $\omega$ 和 $b$ 的偏导并令其为零：

$$\frac{\partial\mathcal{L}}{\partial\omega} = \omega - \sum_{i=1}^{n}\alpha_i y_i x_i = 0 \Rightarrow \omega = \sum_{i=1}^{n}\alpha_i y_i x_i \tag{2.29}$$

$$\frac{\partial\mathcal{L}}{\partial b} = -\sum_{i=1}^{n}\alpha_i y_i = 0 \Rightarrow \sum_{i=1}^{n}\alpha_i y_i = 0 \tag{2.30}$$

将式(2.29)和(2.30)代入拉格朗日函数(2.28)，消去 $\omega$ 和 $b$，得到对偶问题的目标函数：

$$\max_{\alpha}\sum_{i=1}^{n}\alpha_i - \frac{1}{2}\sum_{i=1}^{n}\sum_{j=1}^{n}\alpha_i\alpha_j y_i y_j x_i^T x_j \tag{2.31}$$

$$\text{s.t.} \sum_{i=1}^{n}\alpha_i y_i = 0, \alpha_i \geq 0, i=1,2,...,n$$

根据KKT（Karush-Kuhn-Tucker）条件，最优解必须满足：

$$\begin{cases} \alpha_i \geq 0 \\ y_i(\omega^T x_i + b) - 1 \geq 0 \\ \alpha_i(y_i(\omega^T x_i + b) - 1) = 0 \end{cases} \tag{2.32}$$

KKT条件揭示了SVM的核心性质：对于任意训练样本，要么 $\alpha_i = 0$，该样本对模型无贡献；要么 $y_i(\omega^T x_i + b) = 1$，该样本即为支持向量。这正是SVM模型稀疏性的根本来源——只有少数支持向量决定了最终的超平面。

## 4. 软间隔线性SVM

在实际应用中，数据往往并非完全线性可分，严格满足所有样本点都被正确分类的"硬间隔"条件过于苛刻。为此，软间隔线性SVM放宽了硬间隔中的严格约束，通过引入松弛变量 $\xi_i \geq 0$，允许部分样本点越过分类边界或位于间隔之内，从而得到一个更具弹性的分类模型：

$$y_i(\omega^T x_i + b) \geq 1 - \xi_i, i=1,2,...,n \tag{2.33}$$

相应的优化目标变为在最大化间隔的同时，最小化松弛变量的总和：

$$\min_{\omega,b,\xi}\frac{1}{2}\|\omega\|^2 + C\sum_{i=1}^{n}\xi_i \tag{2.34}$$

$$\text{s.t.} \quad y_i(\omega^T x_i + b) \geq 1 - \xi_i, \xi_i \geq 0, i=1,2,...,n$$

其中 $C > 0$ 为惩罚参数，用于平衡间隔最大化与误分类惩罚之间的关系。$C$ 值越大，模型对误分类的惩罚越重，越倾向于将训练样本正确分类；$C$ 值越小，则允许更多的样本被误分类，以获得更大的分类间隔。

引入松弛变量后，对偶问题形式略有变化，拉格朗日乘子 $\alpha_i$ 的约束变为 $0 \leq \alpha_i \leq C$，KKT条件也相应调整为：

$$\begin{cases} \alpha_i \geq 0, \mu_i \geq 0 \\ y_i(\omega^T x_i + b) - 1 + \xi_i \geq 0 \\ \alpha_i(y_i(\omega^T x_i + b) - 1 + \xi_i) = 0 \\ \mu_i\xi_i = 0 \end{cases} \tag{2.35}$$

其中 $\mu_i$ 是约束 $\xi_i \geq 0$ 对应的拉格朗日乘子。

## 5. 决策函数与概率输出

求解对偶问题得到最优拉格朗日乘子 $\alpha_i^*$ 后，可由式(2.29)计算最优法向量：

$$\omega^* = \sum_{i=1}^{n}\alpha_i^* y_i x_i \tag{2.36}$$

偏置项 $b^*$ 可通过任一支持向量 $(\alpha_s > 0)$ 及其对应样本 $(x_s, y_s)$ 计算：

$$b^* = y_s - (\omega^*)^T x_s = y_s - \sum_{i=1}^{n}\alpha_i^* y_i x_i^T x_s \tag{2.37}$$

对于新样本 $x$，其决策值为：

$$f(x) = (\omega^*)^T x + b^* = \sum_{i=1}^{n}\alpha_i^* y_i x_i^T x + b^* \tag{2.38}$$

式(2.38)揭示了SVM的两个重要特性：

- **稀疏性**：由于KKT条件，只有支持向量对应的 $\alpha_i^* > 0$，其他样本的 $\alpha_i^* = 0$。因此，求和只需对支持向量进行，模型仅由少数支持向量决定。

- **对偶表示**：决策函数仅依赖于训练样本之间的内积 $x_i^T x$，这一性质为后续引入核函数、扩展至非线性分类奠定了基础。

对于线性SVM，式(2.38)中的内积 $x_i^T x$ 即为线性核函数：

$$K(x_i, x) = x_i^T x \tag{2.39}$$

线性核是核函数的最简单形式，计算效率高，适合高维稀疏数据场景。

**与传统SVM直接使用符号函数输出类别标签不同**，本研究采用**Platt缩放（Platt Scaling）**方法，通过Sigmoid函数将决策值映射为概率值：

$$P(y=1|x) = \frac{1}{1 + \exp(-f(x))} = \frac{1}{1 + \exp(-((\omega^*)^T x + b^*))} \tag{2.40}$$

该概率输出为后续图优化模型提供了连续、可微分的节点特征响应值，量化了每个轮廓点属于脊柱点的可能性强度。通过求解上述优化问题，即可获得最优的分类超平面参数，进而实现对脊柱点的概率化识别。线性SVM凭借其坚实的理论基础、清晰的几何解释以及高效的优化算法，成为脊柱点识别任务的理想选择。

## 6. 本研究的SVM实现特点

本研究中SVM的实现具有以下特点：

1. **线性核函数**：采用LibLinear库实现，使用线性核 $K(x_i, x) = x_i^T x$，计算效率高。

2. **三维输入特征**：输入特征包括不对称性（asymmetry）、斜率（slope）和曲率（curvature）三个维度。

3. **特征归一化**：对输入特征进行标准化处理：
   $$x_{norm} = \frac{x - \mu}{\sigma}$$
   其中 $\mu$ 为均值，$\sigma$ 为标准差。

4. **概率输出**：通过Sigmoid函数将决策值映射到 $[0,1]$ 区间，输出脊柱点概率：
   $$P = \frac{1}{1 + e^{-f(x)}}$$

5. **软间隔分类**：采用LibLinear的软间隔实现，通过惩罚参数 $C$ 控制分类边界与误分类的权衡。
